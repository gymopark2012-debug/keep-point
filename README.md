# KeepPoint 프로토타입

KeepPoint는 저장한 링크를 카테고리별로 정리하고, 설명과 마지막 읽기 흐름을 함께 보관하는 서비스입니다.

## 포함 기능

- 카테고리·링크 목록, 메모(설명), 태그
- **PDF**: `pdf-viewer.html`에서 PDF.js로 열고, 페이지·스크롤 위치를 `localStorage`에 저장
- **웹**: 새 탭으로 열기 + **Chrome 확장 프로그램**(`extension/`)으로 스크롤 위치 저장

## 실행 방법

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다.

## 다음 단계 제안

- 확장 ↔ 웹앱 동기화(읽기 요약 표시)
- PDF.js 오프라인 번들
- 로그인·클라우드 동기화
