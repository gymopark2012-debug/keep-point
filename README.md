# KeepPoint 프로토타입

KeepPoint는 저장한 링크를 카테고리별로 정리하고, 설명과 마지막 읽기 흐름을 함께 보관하는 서비스입니다.

## 포함 기능

- 카테고리 목록 및 선택
- 카테고리별 링크 목록
- 링크 상세(제목, 태그, 설명, 마지막 방문 날짜)
- 설명 숨기기/보기
- `읽기 이어서 보기` 클릭 시 마지막 방문 날짜 갱신
- 공유 보기(선택 링크 정보 클립보드 복사)
- 카테고리 추가 / 링크 추가 모달
- 브라우저 로컬 저장소 기반 상태 복구

## 실행 방법

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다.
