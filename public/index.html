        Google · 네이버 간편 로그인 설정

        간편 로그인은 개발자 콘솔 등록과 로컬 서버 실행이 필요합니다. 파일을 더블클릭해서 열면 동작하지 않습니다.

        0. 먼저 로컬 서버로 실행

        프로젝트 폴더에서 아래 중 하나를 실행한 뒤, 표시된 주소로 접속하세요.

        npx --yes serve .
# 또는
python -m http.server 8080

        접속 주소 예: http://localhost:3000/index.html

        1. auth-config.js 에 ID 입력

        프로젝트의 auth-config.js 파일을 열고 발급받은 ID를 넣은 뒤 저장·새로고침하세요.

        2. Google 설정

        1. Google Cloud Console → OAuth 클라이언트 ID → 웹 애플리케이션
        2. 승인된 JavaScript 원본에 아래 주소를 그대로 추가

        http://localhost:3000

        발급된 클라이언트 ID → googleClientId

        3. 네이버 설정

        1. 네이버 개발자센터 → 애플리케이션 → API 설정 → 네이버 로그인 사용
        2. 서비스 URL: 위와 같은 원본 주소 (경로 제외)
        3. Callback URL: 아래 주소를 그대로 등록

        http://localhost:3000/index.html

        Client ID → naverClientId

        배포 후에는 실제 도메인(예: https://your-site.com)으로 같은 항목을 추가해야 합니다.
